# Data Fill 宣传图

目录里包含 6 张宣传图 SVG 和一个本地预览页。

## 文件

- `01-cover.svg`
- `02-pain-points.svg`
- `03-solution.svg`
- `04-features.svg`
- `05-workflow.svg`
- `06-cta.svg`
- `index.html`

## 使用方式

1. 打开 `index.html` 看整体效果。
2. 直接把 SVG 导入 Figma 或 Keynote 再微调。
3. 如果要发插件商店，建议导出为 PNG，尺寸保持 1600x1200。

## 文案定位

- 目标用户：UX / UI / 产品设计师
- 核心卖点：字段化数据填充，点击即填充，文本 / 图片 / AI 一体化
- 主要场景：原型设计、B 端系统设计、Demo / 汇报稿准备
